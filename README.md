# onlineOrder
### 1. Implemented REST API via Spring MVC including reigistration, menu searching and ordering, checkout and etc.
### 2. Utilized Hibernate to access and operate the data storage (menu, restaurants etc.)
### 3. Provided both authentication and authorization via Spring security to protect the application from malicious attachs
### 4. Used the Spring framework core technologies to loosely decouple all the components iin the application
### 5. Build the client side with ReactJS and Ant Design to allow users add items to the shopping cart and place orders.
